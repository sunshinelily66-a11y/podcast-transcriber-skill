# Output Format

## Transcript

- Default location: `transcripts/<audio-stem>.txt`
- Encoding: UTF-8

## Summary

- Default location: `summaries/<audio-stem>.md`
- Recommended sections:
  - Key Points
  - Conclusions
  - Actionable Ideas

## Highlights (Chinese essence summary)

- Default location: `highlights/<audio-stem>.md`
- Recommended sections:
  - `一句话结论`
  - `核心观点`
  - `关键细节与案例`
  - `风险与争议`
  - `可行动启发`
  - `适合二次创作的选题角度`
